**Overview**

A relational SQL database designed to track player statistics and enable performance benchmarking against community averages. The system supports leaderboard-style analytics and comparative performance insights.


**Features**

- Player activity tracking (playtime, win rate, character usage)

- Achievement tracking and rarity analysis

- Leaderboard-style ranking queries

- Player percentile and average benchmarking

- Data cleaning and standardization for reliable analytics


**Technologies**

- SQL (PostgreSQL/MySQL)

- Relational schema design (3NF)

- Aggregate and window functions


**Example Analytics**

- Rank players by total playtime

- Compare individual win rate against global average

- Identify rare achievements

- Compute player percentile rankings
